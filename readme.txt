Contributors: airslim
Tags: comments, spam
Requires at least: 2.0.2
Tested up to: 3.0
Stable tag: trunk


Integrates aTokens badge into posts / pages. 


== Installation ==

1 - unzip the file
2 - copy atokens folder under /wp-content/plugins
3 - login into wordpress admin panel, go to plugins section and activate aTokens plugin
4 - under settings menu, click aTokens menu and customize it
